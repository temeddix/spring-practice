# spring-practice

## macOS

```shell
brew install openjdk
brew install mysql
```

```shell
brew services start mysql
```

```shell
./mvnw spring-boot:run
```

## Windows

```shell
scoop bucket add java
scoop install java/openjdk
scoop install mysql
```

```shell
net start mysql
```

```shell
.\mvnw.cmd spring-boot:run
```
